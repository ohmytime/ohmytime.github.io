# Arduino手动安装开发板


RT, 使用魔法也不行，

那就手动下载，然后把下载的东西逐个放进如下地址，再安装就可以了。

&lt;!--more--&gt;

- Mac 包地址

``` c
/Users/[你的用户名]/Library/Arduino15/staging/packages
```
- Windows 包地址
``` c
/Users/[你的用户名]/Library/Arduino15/staging/packages
```

这样的话，你也可以一次性把所需的文件下载好，做成离线包。
